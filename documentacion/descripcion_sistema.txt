quiero modificar algo el sistema de esta manera profesional para el restaurante casa jualian:

cuando un cliente escribe al whatsap de la empresa - restaurante -asador casa julian automaticamente crea una respuesta de bievenida y saludo inicial y muestra estas 4 opciones: 
"1. QUIERO RESERVAR" : si el cliente elige esta opcion sen seguida le va a pedir una fecha, una hora y numero de comensales. Una vez el cliente haya metido esa informacion el sistema comprobará la disponibilidad el la base de datos de reservas del restaurante y respondera si tienen o No tiene disponibilidad para esa fecha,hora y numero de comensales. En el caso de que NO haya disponibilidad el chatbot se lamentará que no haya disponibilidad y propondrá al cliente si quiere inscribirse en la lista de espera por si hay alguna cancelacion de alguna reserva en el futuro. Si el cliente quiere inscribirse en la lista de espera se le pedirá Nombre, telefono, dni y email y al meter esos datos esa reserva se guardara en la lista de espera de la base de datos del sistema y finalizará la gestion volviendo al menu inicial. En el caso de que SI haya disponibilidad para la fecha , hora y numero de comensales (comprando en la base de datos del sistema de reservas) entonces se le pedirá seguidamente al cliente el Nombre, telefono, dni y email y se guardara en el sistema y se confirmara la reserva, y volverá al menu principal.

"2. LISTA DE ESPERA": si el cliente elige esta opcion el chatbot le va a preguntar por Nombre, telefono, dni y email  y despues de comprobar que existe en el sistema le ha a devolver la posicion en la que esta ese cliente en la lista de espera (indicando cuantos mas hay por delante de ese cliente). una vez que el cliente sabe en que posicion esta en la lista de espera el chatbot le preguntara si quiere esperar o quiere eliminarse de la lista de espera. SI elige eliminar de la lista de espera se borrará ese registro de la base de datos y volvera al menu inicial.

"3.TENGO RESERVA": si el cliente elige esta opcion a continuacion el chatbot va a pedir el nombre, telefono, dni y email y si existe una reserva tal enseguida tiene que ver otros tres botones: VER RESERVA (para ver fecha, hora y nuemro de comensaloes) MOIFICAR RESERVA (para modificar algun dato de reserba si es posible), CANCELAR RESERVA (eliminar reserba y liberar esa disponibilidad). Si no existe reservaa indicará al cliente que no existe una reserva a ese nombre, dni, telefono, email.

"4.PREGUNTAS FRECUENTES": si el cliente elige esta opcion se mostraran una lista con cada una de las preguntas mas frecuentes: carta, ubicacion, horarios, gestion grupos, gestion mascotas, parking, alergias...) que se reponderan con una respuesta automatica o guiando a  https://casajulian.eus/ o enviando un pdf con la carta, horarios y todo.
